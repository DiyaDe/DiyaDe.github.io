我的图床方案

发表于2022-12-04|更新于2022-12-28|杂谈教程
|字数总计:800|阅读时长:2分钟|阅读量:
小懒叨叨

偶然在群里看见群友有很多好看的图片，我就厚着脸皮全部要了过来，为了让的所有人都能看到图片，我便开始了漫长的图床搭建之路

在这里要感谢呆鱼大佬给的方案。

方案一存储量有1G,方案二目前我还不知道有多大，感觉没有上限。

参考教程

村雨遥的博客-免费图床

Typora+Bilibili免费图床|呆鱼

因为vercel域名被污染了，所以需要自备域名使用效果更加。加上picGo是为了更方便的上传图片，不用繁琐的使用命令行，提交。

第一步

在github上创建仓库

名字随便填，但仓库一定要选择为公开（不知道公开的话，默认既可，也可以使用网页翻译一般显示为“公共”二字）。建议给仓库添加一个readm文件。避免在部署到vercel里面是出现错误（一般会出现仓库不能为空的报错。）

第二步

在Vercel官网注册账号，看不懂英文可以使用网页翻译。有账号的话可以忽略这个操作

登录vervcel网站，然后按照图片操作即可。

注意这里选择你刚在git上新建仓库的名字。点击lmport

先写入自己的域名，格式为（name+你的一级域名）例如：tuku.luomengguo.top。这么说吧只带一个点的都是一级域名，这里部署使用的二级域名或者三级域名。然后点击add，再到你购买域名的平台进行解析就可以了。

第三步

到PicGo官网下载软件。下载对应电脑版本即可。

然后开始配置picGo

先到github创建token；以此打开 Settings -> Developer settings -> Personal access tokens，最后点击 generate new token；

滑到最下面选择Developer settings

可能会出现的选项，选择classic

填写好以下信息，其他默认即可。点击创建。

保存好生成的token，这个只能查看一次。

配置 PicGo，打开 图床设置 -> Github 图床；填写相关信息

上传图片在上传区操作即可。

第一步准备工作

到 GitHub 下载插件 (main.exe)

将其解压到你指定文件目录底下 (这个文件目录很重要后面会用到，且文件目录中不能含有空格) 我将其放在了F:\blog\main.exe之下

网页端打开 bilibili 官网，登录自己的账号之后，然后按 F12(或者 Fn + F12 ), 进行如下操作，复制 SESSDATA .

第二步配置Typora

将 main.exe 的目录地址 bilibili 网页端拿到的 SESSDATA 码进行组合，例如：main.exe的目录地址+空格+**token=**+bilibili网页端拿到的SESSDATA码。

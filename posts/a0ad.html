树莓派烧录系统

  1. 下载官方烧录软件，Raspberry Pi Imager，选择自己合适的系统。

  2. 选择要烧录的系统，推荐选择第一个系统，这个系统可以适配树莓派的所有型号

  3. 选择烧录的 SD 卡

  4. 点击设置配置好必要的设置

    可以参考我的配置

  5. 点击烧录，注意：烧录如果长时间都是 0%，可能需要魔法下载系统镜像文件，关于魔法我不多赘述了，想必玩这个的应该都有吧，没有就再说。

ssh 连接树莓派

  1. 下载并安装 ssh 连接工具FinalShell

  2. 获取树莓派本地 IP，下载并安装 IP 扫描工具Advanced IP Scanner （是手机热点的可以忽略这一步，ip 可以直接在热点的连接设备中查看。）

    • 先查看你电脑的本地 IP ，win+R 输入 cmd

    • 将192.168.219.xx粘贴到 Advanced IP Scanner 中进行查询，如果你是第一次登录，显示的名称可能是 IP，需要你每个都尝试一下。如果是你前面设置的主机名称，直接右键复制 IP 就可以了

  3. 连接树莓派
